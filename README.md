# 📄 DOM Implementation Project  

## 📌 Project Overview  
This project demonstrates a simple implementation of the **Document Object Model (DOM)** using HTML, CSS, and JavaScript.  

It allows users to interact with web page elements such as color boxes and input fields dynamically.  

---

## 🎯 Objective  
- To understand how DOM works in web development  
- To manipulate HTML elements using JavaScript  
- To create interactive web elements  

---

## 🛠️ Technologies Used  
- HTML5  
- CSS3  
- JavaScript  

---

## ✨ Features  

### 🎨 Color Boxes  
- Four color options: Red, Blue, Green, Yellow  
- Clicking a box changes its background color  

### 🧑 User Input  
- Input field to enter user name  

### 👋 Greeting Function  
- Displays personalized greeting message  

---

## 📂 Project Structure  

project-folder/
│
├── index.html
└── README.md

---

## ⚙️ How to Run  

1. Download or copy the project files  
2. Open `index.html` in any web browser  
3. Interact with the elements:  
   - Click on color boxes  
   - Enter your name  
   - Click the "Greet" button  

---

## 🔍 How It Works  
- The color boxes use JavaScript to change their own background color when clicked  
- The input field takes user input  
- The button triggers a function that displays a greeting message  

---

## 📸 Output  
- Clicking a color box → Changes only that box color  
- Entering name + clicking Greet → Displays:  
  👉 **Hello, [Your Name]!**  

---

## 📌 Conclusion  
This project helps in understanding the basics of DOM manipulation and user interaction in web development. It is a simple yet effective example for beginners learning JavaScript.
